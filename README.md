# Deep Learning Image Classification Project

A deep learning image classification project built using PyTorch and a custom Residual CNN architecture for CIFAR-style image data. This project demonstrates the complete deep learning workflow including data augmentation, model training, evaluation, visualization, and inference using a fully self-contained setup.

---

## 📌 Project Objective

The goal of this project is to build and evaluate a deep learning image classifier capable of predicting image categories from CIFAR-style datasets.

The project includes:

- Image preprocessing & augmentation
- Custom CNN with residual connections
- Model training and validation
- Training curve visualization
- Evaluation metrics and confusion matrix
- Saved trained model for inference
